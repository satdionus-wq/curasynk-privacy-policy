Curasynk Privacy Policy

Curasynk respects your privacy. This application is designed to work primarily offline on the user's device.

Information Collection

Curasynk does not collect, store, or transmit personal medical data to any centralized server. All application data is stored locally on the user's own device.

Data Storage

  • All records are stored locally on the user device.
  • No automatic cloud backup is performed.
  • No centralized server storage is currently used.
  • Users are responsible for exporting or backing up their own data.

Internet Usage

Curasynk may require internet access only for optional features, application updates, or third-party integrations. Core functionality works offline.

Third-Party Services

Curasynk currently does not sell, rent, or share user medical information with third parties.

Data Security

Because data is stored locally on the user's device, users should protect their devices using passwords, PINs, or biometric security methods.

Children's Privacy

Curasynk is not intended for children under 13 years of age.

Changes to This Policy

This Privacy Policy may be updated in the future as application features evolve.
